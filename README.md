# YouTube Queue - Yoga Extension

This extension saves YouTube videos to the `Yoga` collection and displays them on a dedicated page.

## Installation

1. Clone or download the `youtubeQueYoga` folder.
2. Open Microsoft Edge and go to `edge://extensions/`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the `youtubeQueYoga` folder.

## Usage

1. Go to a YouTube video page.
2. Right-click on a video or link.
3. Select "Add to Yoga" from the context menu.
4. Visit your GitHub Pages website at `https://yourusername.github.io/youtubeque/yoga.html` (update the URL in `manifest.json`) to view the Yoga collection.
5. Use the Clear All button on the Yoga page to remove the full queue.

## Notes

- This is a separate extension from the main "YouTube Queue" extension.
- It stores videos under the `yoga` key.
- Run both extensions separately if you want one for home queue and one for yoga.
